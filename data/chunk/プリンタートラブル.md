# プリンタートラブルシューティング

## プリンターが「オフライン」と表示される

### 確認手順
1. プリンター本体の電源が入っているか確認
2. LANケーブルが接続されているか確認
3. プリンター本体のディスプレイにエラーが表示されていないか確認

### PC側の対処
1. 設定 → Bluetooth とデバイス → プリンターとスキャナー
2. 対象プリンターをクリック
3. 「印刷キューを開く」をクリック
4. 「プリンター」メニュー → 「プリンターをオフラインで使用する」のチェックを外す
5. それでも解決しない場合は、プリンタースプーラーをリセット：
   - コマンドプロンプト（管理者）を開く
   - `net stop spooler` を実行
   - `net start spooler` を実行

## 印刷が詰まる・出てこない

### スプーラーのクリア
1. コマンドプロンプト（管理者）を開く
2. `net stop spooler` を実行
3. `del /Q /F /S "%systemroot%\System32\spool\PRINTERS\*.*"` を実行
4. `net start spooler` を実行

### 紙詰まりの対処
1. プリンターの前面カバーを開ける
2. 詰まった用紙をゆっくり引き出す（無理に引っ張らない）
3. 紙片が残っていないか確認
4. カバーを閉じて再度印刷

## トナー交換の目安
- 「トナー残量わずか」表示：まだ数百枚は印刷可能
- 「トナーを交換してください」表示：すぐに交換が必要
- 交換用トナーは総務部に在庫あり。型番を確認して申請
