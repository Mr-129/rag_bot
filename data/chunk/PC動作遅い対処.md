# 社内PC動作が遅い場合の対処法

## 基本チェック

### ステップ1：タスクマネージャーで原因を特定
1. Ctrl + Shift + Esc でタスクマネージャーを開く
2. 「プロセス」タブでCPU・メモリの使用率を確認
3. 異常に高いプロセスがあればチェック

### よくある原因と対処

#### CPU使用率が高い（90%以上）
- Windows Updateが裏で動いている → 完了するまで待つ
- ウイルススキャン実行中 → スキャン完了を待つ
- 特定アプリが暴走 → そのアプリを右クリックして「タスクの終了」

#### メモリ使用率が高い（85%以上）
- ブラウザのタブを開きすぎ → 不要なタブを閉じる
- 重いアプリを複数起動 → 不要なアプリを閉じる
- メモリが8GB以下 → IT部門に増設を相談

#### ディスク使用率が100%
- Windows Search Indexerが動作中 → しばらく待つ
- SysmainサービスのプリフェッチでI/O過多 → 再起動で改善する場合が多い

### ステップ2：スタートアップの最適化
1. タスクマネージャー →「スタートアップ」タブ
2. 不要な自動起動アプリを「無効」に設定
3. 必須以外のスタートアップアプリ：
   - 無効にして良い：Adobe Updater, iTunes Helper, Skype
   - 無効にしてはいけない：Defender, FortiClient, OneDrive

### ステップ3：再起動
PC を定期的に再起動することでメモリリークやキャッシュの肥大化を解消できます。
- **推奨**: 週に1回は「シャットダウン」（再起動）を実施
- 「スリープ」だけではメモリが解放されない場合がある

## それでも改善しない場合
- IT部門にPC交換またはメモリ増設を依頼
- SSD未搭載の古いPCはSSD換装で劇的に改善する場合あり
