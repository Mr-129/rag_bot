# Windows Updateの管理と対処法

## 社内の更新ポリシー
- 毎月第2火曜日（日本時間水曜日）にMicrosoftから更新プログラムがリリース
- 社内ではWSUS（Windows Server Update Services）で配信を管理
- 配信スケジュール：リリースの1週間後に全社展開

## 手動での更新確認
1. 設定 → Windows Update
2. 「更新プログラムのチェック」をクリック
3. 利用可能な更新がある場合はダウンロード・インストール
4. 再起動が必要な場合は業務終了時に実施

## 更新後にPCが起動しない場合

### セーフモードで起動
1. 電源ボタンを長押しして強制電源OFF（3回繰り返す）
2. 「自動修復」画面が表示される
3. 「詳細オプション」→「スタートアップ設定」→「再起動」
4. 「4) セーフモードを有効にする」を選択

### 更新のロールバック
1. セーフモードで起動
2. 設定 → Windows Update → 更新の履歴
3. 「更新プログラムをアンインストールする」をクリック
4. 問題の更新を選択して「アンインストール」

## ブルースクリーン（BSOD）が発生した場合
1. エラーコードを記録（例：IRQL_NOT_LESS_OR_EQUAL）
2. PCを再起動してセーフモードで起動
3. デバイスマネージャーでドライバーを確認
4. 最近インストールしたドライバーをロールバック
5. 解決しない場合はIT部門に連絡（エラーコードを伝える）

## 更新の一時停止
業務上やむを得ない場合のみ、最大5週間まで一時停止可能：
1. 設定 → Windows Update
2. 「更新の一時停止」で期間を選択
※ セキュリティリスクが高まるため、極力一時停止しないでください
