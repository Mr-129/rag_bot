# ディスク容量不足の対処法

## 症状
「ディスク領域不足」の警告が表示される、またはアプリケーションの動作が遅くなる。

## 確認方法
1. エクスプローラー → 「PC」を開く
2. Cドライブの空き容量を確認
3. 空き容量が10GB未満の場合は対処が必要

## 対処手順（効果が大きい順）

### 1. ディスクのクリーンアップ
1. エクスプローラーでCドライブを右クリック →「プロパティ」
2. 「ディスクのクリーンアップ」をクリック
3. 「システムファイルのクリーンアップ」をクリック
4. 以下にチェックを入れる：
   - Windows Updateのクリーンアップ
   - 一時ファイル
   - ごみ箱
   - 配信の最適化ファイル
5. 「OK」→「ファイルの削除」

### 2. 一時ファイルの削除
1. 設定 → システム → ストレージ
2. 「一時ファイル」をクリック
3. 削除対象を選択して「ファイルの削除」

### 3. 不要アプリの削除
1. 設定 → アプリ → インストールされているアプリ
2. サイズ順でソート
3. 不要なアプリを「アンインストール」

### 4. OneDriveのファイルオンデマンド活用
1. ローカルに保持する必要のないファイルを右クリック
2. 「空き領域を増やす」を選択
3. ファイルはクラウドに残り、必要時にダウンロードされる

## 注意事項
- `C:\Windows\WinSxS`フォルダは手動で削除しないこと
- `C:\Users\ユーザー名\AppData`内のファイルは慎重に扱うこと
- 空き容量が5GB未満の場合はIT部門に連絡して拡張を検討
