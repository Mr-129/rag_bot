# PCがフリーズ・ブルースクリーンになった場合

## 画面が固まった（フリーズ）場合

### マウスが動く場合
1. Ctrl + Alt + Delete を押す
2. 「タスクマネージャー」を選択
3. 「応答なし」のアプリを選択して「タスクの終了」
4. 保存していなかったデータは失われる可能性あり

### マウスも動かない場合
1. 5〜10分待って回復しないか確認
2. 回復しない場合、電源ボタンを10秒間長押しで強制終了
3. 30秒待ってから再度電源を入れる
4. 「Windowsが正しく終了しませんでした」と表示されたら「通常起動」

## ブルースクリーン（BSOD）が表示された場合

### 記録すべき情報
- エラーコード（例：CRITICAL_PROCESS_DIED、IRQL_NOT_LESS_OR_EQUAL）
- QRコードの下に表示されるSTOPコード
- 発生した状況（何をしていたか、どのアプリを使っていたか）

### 対処手順
1. 自動的に再起動される（または電源ボタンで再起動）
2. 正常に起動したら、イベントビューアーでエラーログを確認
3. 直近にインストールしたソフトやドライバーがあれば削除を検討
4. Windows Updateがすべて適用されているか確認

### 頻繁に発生する場合
以下の情報を添えてIT部門に連絡：
- エラーコード
- 発生頻度（毎日/週1回/月1回など）
- 特定の操作で発生するか
- 発生し始めた時期

## 予防策
- 定期的な再起動（週1回推奨）
- Windows Updateを最新に保つ
- 不要なアプリを閉じてメモリを節約
- PCの通気口を塞がない（熱暴走防止）
