# Outlookメール設定方法

## 新規アカウント設定
1. Outlookを起動する
2. 「ファイル」→「アカウントの追加」をクリック
3. 社内メールアドレスを入力して「接続」をクリック
4. パスワードを入力
5. 多要素認証（MFA）の承認画面が表示されたら、スマートフォンのAuthenticatorアプリで承認
6. 「アカウントが正常に追加されました」と表示されたら完了

## 共有メールボックスの追加
1. Outlookの「ファイル」→「アカウント設定」→「アカウント設定」
2. 既存のExchangeアカウントを選択して「変更」
3. 「詳細設定」→「詳細設定」タブ
4. 「追加」ボタンをクリック
5. 共有メールボックスの名前を入力して「OK」

## メール署名の設定
1. 「ファイル」→「オプション」→「メール」→「署名」
2. 「新規作成」をクリック
3. 署名名を入力（例：「標準」）
4. 署名本文を入力
5. 「新しいメッセージ」と「返信/転送」それぞれに使用する署名を選択

## 自動仕分けルールの作成
1. 「ホーム」→「ルール」→「仕分けルールと通知の管理」
2. 「新しい仕分けルール」をクリック
3. 条件（差出人、件名のキーワードなど）を設定
4. アクション（フォルダへ移動、フラグを付けるなど）を選択
5. 「完了」をクリック
