# 社用スマートフォンのメール設定

## 前提条件
- 会社支給のスマートフォンであること
- Microsoft Intune（MDM）が登録済みであること
- 画面ロック（PIN 6桁以上）が設定されていること

## iPhoneの場合

### Outlookアプリのインストール
1. App StoreでMicrosoft Outlookを検索してインストール
2. Outlookを起動
3. 社内メールアドレスを入力 →「アカウントの追加」
4. パスワードを入力
5. MFA認証を完了
6. 通知の許可を設定

### 標準メールアプリでの設定
1. 設定 →「メール」→「アカウント」→「アカウントを追加」
2. 「Microsoft Exchange」を選択
3. メールアドレスとパスワードを入力
4. 同期する項目（メール、カレンダー、連絡先）を選択

## Androidの場合

### Outlookアプリのインストール
1. Google PlayでMicrosoft Outlookを検索してインストール
2. Outlookを起動
3. 社内メールアドレスを入力
4. パスワードを入力してMFA認証を完了

## セキュリティ要件
- 画面ロック：PIN 6桁以上（生体認証も可）
- 暗号化：有効であること
- OSバージョン：最新から2世代前まで対応
- Jailbreak/Root化された端末は接続不可

## スマートフォンの紛失・盗難時
1. 直ちにIT部門に連絡
2. IT部門がMicrosoft Intuneでリモートワイプ（データ消去）を実行
3. 紛失届を提出
4. 代替端末の手配
