# 経費精算システムの使い方

## アクセス方法
社内ポータル →「経費精算」、またはブックマークからアクセス

## 交通費精算
1. 「新規申請」→「交通費精算」を選択
2. 利用日、路線、区間を入力
3. ICカード利用の場合は運賃が自動計算
4. 定期券区間が自動控除される
5. 「申請」をクリック

## 出張経費精算
1. 「新規申請」→「出張精算」を選択
2. 出張期間と目的を入力
3. 費目ごとに金額を入力：
   - 交通費（新幹線、飛行機など）
   - 宿泊費
   - 日当
   - その他（タクシー、駐車場など）
4. 領収書の画像を添付（スマートフォンで撮影可）
5. 「申請」をクリック

## 立替経費精算
1. 「新規申請」→「立替精算」を選択
2. 購入物、金額、購入先を入力
3. 領収書の画像を添付
4. 予算コードを選択
5. 「申請」をクリック

## 領収書の取り扱い
- 電子保存が原則（スマートフォンの撮影でOK）
- 撮影時の注意：
  - 金額、日付、宛名が鮮明に読めること
  - 影や反射がないこと
  - トリミング不要で全体が写っていること
- 紙の領収書は申請承認後1年間保管

## 承認フロー
1. 申請者が申請
2. 直属の上長が承認
3. 経理部門が確認・精算
4. 翌月の給与に精算額が上乗せ
