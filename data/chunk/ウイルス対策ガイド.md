# ウイルス対策ソフトの確認と対応

## 社内環境
- 全社PCにMicrosoft Defender for Endpointを導入
- リアルタイム保護が常時有効
- 定義ファイルは自動更新

## ウイルスが検出された場合
### 自動対応
1. Defenderが脅威を自動検出し、ポップアップ通知が表示される
2. 多くの場合、自動で「検疫」（隔離）される
3. 「操作が必要です」と表示された場合は以下を実施

### 手動対応
1. Windowsセキュリティを開く
2. 「ウイルスと脅威の防止」→「脅威の履歴」
3. 検出された脅威を確認
4. 推奨されるアクション（削除/検疫）を実行
5. IT部門に検出内容を報告

## フルスキャンの実行
通常は自動スキャンで十分ですが、手動でフルスキャンを行う場合：
1. Windowsセキュリティ →「ウイルスと脅威の防止」
2. 「スキャンのオプション」をクリック
3. 「フルスキャン」を選択
4. 「今すぐスキャン」をクリック
5. 完了まで1〜2時間かかる場合がある

## フィッシングメールへの対応
1. 不審なメールのリンクはクリックしない
2. 添付ファイルは開かない
3. Outlookで該当メールを右クリック →「レポート」→「フィッシング」
4. 誤ってリンクをクリックした場合：
   - パスワードを入力してしまった → 直ちにパスワード変更 + IT部門に報告
   - ファイルをダウンロードしてしまった → ネットワークを切断 + IT部門に連絡
