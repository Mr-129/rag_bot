# ソフトウェアインストール申請手順

## 概要
社内PCへのソフトウェアインストールには管理者権限が必要です。承認済みソフトはSoftware Centerから、それ以外は申請が必要です。

## Software Centerからのインストール（承認済みソフト）
1. スタートメニューから「Software Center」を起動
2. 「アプリケーション」タブで利用可能なソフト一覧を確認
3. インストールしたいソフトを選択
4. 「インストール」をクリック
5. インストール完了後、再起動が必要な場合がある

### 利用可能なソフトウェア例
- Microsoft 365 Apps
- Adobe Acrobat Reader / Pro
- 7-Zip
- Chrome / Firefox
- Visual Studio Code
- WinSCP / TeraTerm
- Slack

## 未承認ソフトのインストール申請
1. 社内ポータル →「IT申請」→「ソフトウェアインストール申請」
2. 以下の情報を入力：
   - ソフトウェア名とバージョン
   - 利用目的
   - ライセンス情報（フリーソフト/有料）
   - 公式ダウンロードURL
3. 上長の承認を取得
4. IT部門による安全性審査（1〜3営業日）
5. 承認後、IT部門がリモートでインストールまたは一時的な管理者権限を付与

## 禁止されているソフトウェア
- P2Pファイル共有ソフト（BitTorrent等）
- 非公式のリモートアクセスツール
- ライセンス違反のソフトウェア
- 暗号通貨マイニングソフト

## 注意事項
- フリーソフトでも業務利用にはライセンス確認が必要
- インストール時にバンドルソフト（不要なツールバー等）を一緒にインストールしないよう注意
